### Example 1: _Using Big Data Pool_

    defaultDataLakeStorageFilesystem: '<defaultDataLakeStorageFilesystem>'
    name: 'swbdp001'
    sqlAdministratorLogin: 'synwsadmin'

          maxNodeCount: 5
