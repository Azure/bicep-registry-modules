## Parameters

**Required parameters**

| Parameter | Type | Description |
| :-- | :-- | :-- |
| [`name`](#parameter-name) | string | Name of the Log Analytics Cluster. |

**Optional parameters**

| Parameter | Type | Description |
| :-- | :-- | :-- |
| [`associatedWorkspaces`](#parameter-associatedworkspaces) | array | List of workspace resource IDs to associate with the cluster. Log Analytics workspaces in the specified list will use this cluster as their data destination. |
| [`billingType`](#parameter-billingtype) | string | The billing type of the cluster. If not specified, the default is 'CapacityReservation'. The cluster can be used for free for the first 30 days if 'PayAsYouGo' is selected. |
| [`capacityReservationProperties`](#parameter-capacityreservationproperties) | object | Properties for capacity reservation. |
| [`diagnosticSettings`](#parameter-diagnosticsettings) | array | The diagnostic settings of the service. |
| [`enableTelemetry`](#parameter-enabletelemetry) | bool | Enable/Disable usage telemetry for module. |
| [`isAvailabilityZonesEnabled`](#parameter-isavailabilityzonesenabled) | bool | Whether availability zones are enabled. |
| [`isDoubleEncryptionEnabled`](#parameter-isdoubleencryptionenabled) | bool | Whether double encryption is enabled. |
| [`keyVaultProperties`](#parameter-keyvaultproperties) | object | Properties for configuring key vault for double encryption. |
| [`location`](#parameter-location) | string | Location for all resources. |
| [`lock`](#parameter-lock) | object | The lock settings of the service. |
| [`managedIdentities`](#parameter-managedidentities) | object | The managed identity definition for this resource. Enables system-assigned and/or user-assigned identities. |
| [`replication`](#parameter-replication) | object | Configuration for cluster replication. When enabled, the cluster can replicate data to another region. |
| [`roleAssignments`](#parameter-roleassignments) | array | Array of role assignments to create. |
| [`size`](#parameter-size) | int | The cluster capacity in Capacity Units (CUs). A CU represents a reservation of 100 GB/day ingestion and 400 GB storage. Minimum of 1000 CUs and maximum of 2000 CUs. |
| [`tags`](#parameter-tags) | object | Tags of the resource. |

### Parameter: `name`

Name of the Log Analytics Cluster.

- Required: Yes
- Type: string

### Parameter: `associatedWorkspaces`

List of workspace resource IDs to associate with the cluster. Log Analytics workspaces in the specified list will use this cluster as their data destination.

- Required: No
- Type: array
- Default: `[]`

### Parameter: `billingType`

The billing type of the cluster. If not specified, the default is 'CapacityReservation'. The cluster can be used for free for the first 30 days if 'PayAsYouGo' is selected.

- Required: No
- Type: string
- Default: `'CapacityReservation'`
- Allowed:
  ```Bicep
  [
    'CapacityReservation'
    'PayAsYouGo'
  ]
  ```

### Parameter: `capacityReservationProperties`

Properties for capacity reservation.

- Required: No
- Type: object
- Default: `{}`
