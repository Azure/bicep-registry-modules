# Azure Kubernetes Service (AKS) Managed Cluster Agent Pools `[Microsoft.ContainerService/managedClusters/agentPools]`

This module deploys an Azure Kubernetes Service (AKS) Managed Cluster Agent Pool.

- [Outputs](#Outputs)
