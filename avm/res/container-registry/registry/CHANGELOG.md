# Changelog

The latest version of the changelog can be found [here](https://github.com/Azure/bicep-registry-modules/blob/main/avm/res/container-registry/registry/CHANGELOG.md).

## 0.10.0

### Changes

- Added `roleAssignmentMode` parameter to specify the registry permissions role assignment mode.
- Added `autoGeneratedDomainNameLabelScope` parameter to control the scope of the auto-generated domain name label.
- Added new ABAC roles.

### Breaking Changes

- None
